Guide de Nooblety 1.0 Help

Dossier de configuration

Information importante

Si vous remplissez les conditions minimales, vous pourrez jouer au jeu, mais vous ne pourrez pas profiter de toutes les fonctionnalités. Réduire les paramètres graphiques peut améliorer les performances, mais cela peut également rendre le jeu moins agréable à jouer.

Configuration requise

  • Système d'exploitation: Windows 10/11 (64 bits) / Linux (64 bits) / macOS (64 bits)

  • Processeur: Intel Core i5-8400 3.1GHz / AMD Ryzen 5 3600 3.6GHz

  • Espace disque: 5Go

  • Mémoire vive: 8 Go

  • Carte graphique: NVIDIA GeForce RTX 3060 / AMD Radeon RX 6600 XT

  • DirectX: Version 11

  • Mémoire graphique: 2 Go

Information complémentaire

Un chipset graphique ne suffit pas pour jouer à Nooblety. Vous devez avoir une carte graphique dédiée.

Fichiers de configuration

  • ProjectSettings/EditorSettings.asset: Ce fichier contient des paramètres liés à l'éditeur Unity, tels que les préférences de l'interface utilisateur, les paramètres de couleur, et les raccourcis clavier personnalisés.

  • ProjectSettings/InputManager.asset: Ce fichier spécifie les entrées de l'utilisateur, telles que les boutons, les axes, et les touches du clavier, et permet de les personnaliser.

  • ProjectSettings/TagManager.asset: Il permet de définir les balises (tags) et les couches (layers) utilisées dans le jeu. Les balises sont couramment utilisées pour identifier des objets spécifiques, tandis que les couches sont utilisées pour déterminer les interactions entre les objets.

  • ProjectSettings/Physics2DSettings.asset: Ce fichier de configuration contient des paramètres physiques pour les éléments en 2D, tels que la gravité, les matériaux de collision, et les couches de collision.

  • ProjectSettings/QualitySettings.asset: Il permet de configurer les niveaux de qualité graphique du jeu, notamment les résolutions d'écran, les paramètres d'anti-aliasing, et d'autres paramètres graphiques.

  • ProjectSettings/PlayerSettings.asset: Ce fichier est crucial pour configurer les paramètres du lecteur (build player settings) tels que la plate-forme de destination, la version du moteur de jeu, les icônes de l'application, et les paramètres de compression des textures.

  • ProjectSettings/ProjectSettings.asset: Il contient divers autres paramètres globaux du projet, tels que la définition de l'ordinateur cible, les paramètres d'importation des modèles 3D, les paramètres audio, etc.

  • ProjectSettings/PackageManagerSettings.asset: Ce fichier gère les paramètres liés au gestionnaire de paquets Unity, qui permet de gérer les packages et les extensions utilisés dans le projet.

  • ProjectSettings/EditorBuildSettings.asset: Il détermine quels sont les scènes incluses dans la compilation du projet pour chaque plate-forme.

  • ProjectSettings/VFXLibrarySettings.asset: Si votre projet utilise des effets visuels (Visual Effects), ce fichier permet de configurer certains paramètres liés aux effets visuels.
